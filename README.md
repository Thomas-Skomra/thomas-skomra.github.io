# Personal CV Website

This is my **personal CV (Curriculum Vitae) website** built using modern web technologies.

## Technologies Used

- **Angular** — Frontend framework for building a robust, component-based SPA (Single Page Application).
- **Tailwind CSS** — Utility-first CSS framework used for rapid and customizable styling.
- **PrimeNG** — A rich set of open-source UI components for Angular, used to create interactive elements and improve user experience.

## Features

- Responsive design that looks great on desktop and mobile devices.
- Interactive UI components powered by PrimeNG (e.g., dropdowns, cards, modals).
- Language switching support.
- Light/Dark theme switching support
- Custom styling with Tailwind CSS for a modern, clean look.
- Easily extensible and maintainable Angular architecture.
